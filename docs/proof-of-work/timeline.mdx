The **Timeline** page provides a minute-by-minute visual representation of employee activity throughout the day. It helps managers understand how employees spend their time and identify productivity patterns.

[https://app.trackpilots.com/timeline](https://app.trackpilots.com/timeline)

---

## 📌 Overview

The Timeline module displays a graphical timeline of employee activity categorized into **Productive, Neutral, Unproductive, and Idle** segments. Each block represents tracked activity based on apps, websites, and system usage.

<Callout type="info">
📋 Timeline data is generated automatically when employees use the Trackpilots desktop app. If the Trackpilots app is not running, activity blocks will not appear in the timeline.
</Callout>

---

## 👥 Team & User Filters

Admins can filter timeline data using the following options:

| Filter        | Description                              |
| ------------- | ---------------------------------------- |
| Team Selector | View activity for a specific team        |
| User Selector | View timeline for an individual employee |
| Date Selector | View timeline for a specific day         |

---

## ⏰ Timeline Visualization

The timeline is displayed in hourly segments (12 AM to 11 PM) with colored activity blocks.

| Activity Type   | Meaning                               |
| --------------- | ------------------------------------- |
| 🟢 Productive   | Work-related apps and websites        |
| 🟡 Neutral      | Research or general browsing          |
| 🔴 Unproductive | Non-work or distracting apps/websites |
| ⚪ Idle          | No keyboard/mouse activity detected   |

---

## 🔄 How Timeline Data Sync Works

1. Employee uses the Trackpilots desktop app
2. App tracks active windows and websites
3. Productivity rules classify each activity
4. Timeline blocks are generated in real time
5. Data is synced to the Trackpilots dashboard
