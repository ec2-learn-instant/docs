## 📌 Overview

The **Pricing** page displays the complete reseller pricing structure for the Trackpilots Engine — the infrastructure that powers all child organisations under your account.

Pricing is broken into five sections:

| Section | Description |
|---|---|
| 👥 User Tier Pricing | Cost per user per month based on total active user count |
| 📊 Fair Usage Limits | Per-user throttling thresholds applied across organisations |
| 📈 Overage Rates | Charges when usage exceeds fair usage limits |
| 📦 Optional Services | Add-on services available beyond the base plan |
| 🗄️ Data Retention Policies | Storage duration and costs to extend data retention |

👉 Direct link: https://app.trackpilots.com/resellers/pricing

---

## 💱 Currency Toggle

All prices are shown in either **INR (₹)** or **USD ($)**. Use the toggle in the top-right corner of the page to switch between currencies. The conversion applies instantly across all tables.

---

## 👥 User Tier Pricing

Pricing is tiered based on your **total active user count** across all child organisations. The more users you have, the lower the per-user cost.

| Tier | User Range | Price per User / Month |
|---|---|---|
| Starter | 1 – 50 | Shown in dashboard |
| Growth | 51 – 200 | Shown in dashboard |
| Scale | 201 – 500 | Shown in dashboard |
| Enterprise | 500+ | Contact Sales |

<Callout type="info">
📞 **Enterprise Pricing**

For user counts above the maximum tier, the pricing table shows a **Contact Sales** badge. Reach out to **team@trackpilots.com** to discuss custom enterprise rates.
</Callout>

User count is calculated as the **sum of active users** across all organisations under your reseller account.

---

## 📊 Fair Usage Limits

Fair usage limits define the maximum resource consumption **per user per month** before overage charges apply. These limits are enforced uniformly across all your child organisations.

| Metric | Limit | Unit |
|---|---|---|
| Screenshots | Shown in dashboard | Per user / month |
| App Tracking Events | Shown in dashboard | Per user / month |
| Activity Events | Shown in dashboard | Per user / month |
| Storage | Shown in dashboard | GB per user |

<Callout type="info">
ℹ️ **Per-User Throttling**

Limits apply at the individual user level, not at the organisation level. A single high-usage user cannot consume another user's quota.
</Callout>

Any usage notes or special conditions for a metric are displayed as an italic info tag (ℹ️) next to the metric name in the dashboard.

---

## 📈 Overage Rates

When a user's usage exceeds the fair usage limit for a given metric, overage charges apply at the following rates:

| Type | Rate (INR/USD) | Per |
|---|---|---|
| Extra Screenshots | Shown in dashboard | Per screenshot |
| Extra App Tracking | Shown in dashboard | Per event |
| Extra Storage | Shown in dashboard | Per GB |

Overage charges are calculated at the end of each billing cycle and added to your invoice automatically.

<Callout type="warning">
⚠️ **Monitor Usage Proactively**

Check the Usage Analytics page regularly to spot organisations approaching their limits before overage charges accumulate.
</Callout>

|---|---|---|---|

## 🗄️ Data Retention Policies

By default, Trackpilots stores tracking data for a standard retention window. Extended retention is available as a paid add-on:

| Policy | Default Retention | Extended Cost |
|---|---|---|
| Screenshot Data | Shown in dashboard | Shown in dashboard |
| Activity Logs | Shown in dashboard | Shown in dashboard |
| App Tracking Data | Shown in dashboard | Shown in dashboard |

<Callout type="info">
🔒 **Data Ownership**

All data stored within your reseller account and child organisations is owned by you. Trackpilots does not share or sell data to third parties.
</Callout>

---

## 🧾 Billing Notes

- All prices are **exclusive of applicable taxes** (GST, VAT, etc.)
- Billing is calculated monthly based on active user count at the end of each cycle
- Overage charges are added to the following month's invoice
- Annual plans may be available at a discounted rate — contact your account manager
