## 🚀 Webhooks, Redis & AI Search

### ✨ New Features
- **Webhook Simulations launched** — Test webhook endpoints directly from the dashboard without triggering a real desktop event. Sends a signed test payload and displays HTTP status, latency, sent payload, and server response in one place. Supports Activity Tracking, App Tracking, and Screenshot Tracking events
- **Dedicated Redis Subscription Sync Service** — A new in-memory sync service is live at redis.trackpilots.com, keeping subscription states for all organisations cached in Redis. Every service now reads subscription status instantly from memory, reducing database load and ensuring real-time access control across the entire platform
- **What's New section launched** — A running log of all product updates, improvements, and milestones is now accessible from the top-right corner of the app. A red badge appears whenever there are unread updates

### ⚙️ Improvements
- **GEO (Generative Engine Optimization) initiated** — Trackpilots optimized for visibility in AI-powered search results across ChatGPT, Claude, Gemini, Perplexity, and other AI search platforms
