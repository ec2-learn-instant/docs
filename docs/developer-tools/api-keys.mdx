## Overview

Trackpilots uses **API Keys** to securely authenticate API requests.  
Each API key is unique and should be kept confidential.

You can generate and manage your API keys from the **Developer Tools** section in the Trackpilots dashboard.

---

## Create an API Key

Follow the steps below to generate a new API key:

1. Go to **Developer Tools** in your Trackpilots dashboard.
2. Select **API Keys** from the menu.  
   üëâ Direct link:  
   `https://app.trackpilots.com/developer-tools/api-keys`
3. Click the **Create API Key** button located at the **top-right corner**.
4. Enter a **name** for your API key.
5. Click **Create**.

üéâ Your API key will be created successfully.
