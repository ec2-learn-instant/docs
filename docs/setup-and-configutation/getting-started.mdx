This guide helps you quickly set up **Trackpilots**, invite your team members, and start monitoring productivity and time tracking in your organization.

---

# 🧭 Overview of Setup Flow

| Step | Action                   | Description                                         |
| ---- | ------------------------ | --------------------------------------------------- |
| 1️⃣  | Create Admin Account     | Register your organization and get dashboard access |
| 2️⃣  | Access Dashboard         | Manage employees and monitoring settings            |
| 3️⃣  | Invite Employees         | Invite via link or add manually                     |
| 4️⃣  | Employee Accepts Invite  | Employee creates their account                      |
| 5️⃣  | Download Trackpilots App | Install desktop tracking agent                      |
| 6️⃣  | Employee Login           | Start tracking activity                             |
| 7️⃣  | Data Sync                | View productivity data in dashboard                 |

---

# 📝 Step 1: Create an Admin Account

To start using Trackpilots, you need to create an **Admin (Organization Owner) account**.

👉 **Sign up here:** [https://trackpilots.com/sign-up](https://trackpilots.com/sign-up)

### ✅ Steps

1. Enter your organization name, email, and password.
2. Submit the registration form.
3. Verify your email if required.

### 🎯 Result

* Your **organization workspace is created**.
* You will receive **Admin Dashboard access**.

---

# 📊 Step 2: Access the Admin Dashboard

After successful signup, you can access the **Trackpilots Admin Dashboard**.

### From the dashboard, you can:

* 👥 Manage team members
* 📈 View productivity analytics
* 🗓️ Monitor attendance and time tracking
* ⚙️ Configure monitoring policies and settings

The dashboard is the **central control panel** for your organization.

---

# 👥 Step 3: Invite or Add Employees

You can add employees to Trackpilots in **two ways**.

👉 **Manage Employees Page:** [https://app.trackpilots.com/settings/manage-employee](https://app.trackpilots.com/settings/manage-employee)

---

## 🔗 Option A: Invite via Invite Link

| Action                | Description                                                                 |
| --------------------- | ------------------------------                                              |
| Click Invite Employee | Enter employee email and generate invitation                                |
| System Sends Email    | Trackpilots automatically sends the invite link to the employee’s email 📩  |
| Employee Registers    | Employee opens the email and creates their account                          |

### ✅ Benefits

* Fast onboarding
* No manual data entry
* Scalable for large teams

---

## ✍️ Option B: Add Employee Manually

| Field      | Description                        |
| ---------- | ---------------------------------- |
| Name       | Employee full name                 |
| Email      | Employee work email                |
| Role       | Employee role (Admin/User/Manager) |
| Password   | Employee temporary password        |

### ✅ Use Case

* When employees cannot register themselves
* For controlled enterprise onboarding

---

# 📩 Step 4: Employee Accepts Invitation

Once invited, employees must complete their account setup.

### Steps for Employee

1. Open the invite link received via email.
2. Set up their password and profile.
3. Complete account registration.

### 🎯 Result

* Employee will appear in the **Trackpilots Employee List** in the dashboard.

---

# 💻 Step 5: Download and Install Trackpilots App

To start tracking activity, employees must install the **Trackpilots Desktop Agent**.

👉 **Download Link:** [https://trackpilots.com/downloads](https://trackpilots.com/downloads)

### Supported Platforms

| Platform | Status       |
| -------- | -----------  |
| Windows  | ✅ Supported |
| macOS    | ✅ Supported |
| Linux    | ✅ Supported |

### Installation Steps

1. Download the installer.
2. Run the setup file.
3. Complete installation on employee system.

---

# 🔐 Step 6: Employee Login to Trackpilots App

After installation:

### Steps

1. Open the Trackpilots desktop app.
2. Login using employee credentials.
3. Grant required permissions (if prompted).

### 🎯 Result

* Trackpilots agent starts tracking work activity automatically.

---

# 🔄 Step 7: Data Sync to Admin Dashboard

Once the employee logs in:

### Automatically Synced Data

| Data Type              | Description                 |
| ---------------------- | --------------------------- |
| ⏱️ Work Time           | Active and idle time        |
| 📸 Screenshots         | Periodic screen captures    |
| 🌐 App & Website Usage | Productivity classification |
| 🧾 Attendance          | Check-in, check-out, breaks |

### Benefits

* 📊 Real-time productivity monitoring
* 📈 Historical analytics and reports
* 👥 Team performance insights
