The **Timesheet** page provides a detailed breakdown of employee working hours, productivity levels, idle time, and attendance records. It helps managers analyze work patterns and ensure accurate time tracking.

[https://app.trackpilots.com/timesheet](https://app.trackpilots.com/timesheet)

---

## 📌 Overview

The Timesheet module automatically logs employee work sessions when they use the Trackpilots desktop app. It records **In Time, Out Time, Total Worked Hours, Active Hours, Idle Hours, and Productivity Metrics** for each day.


<Callout type="info">
📋 Timesheet data is generated automatically when employees log in and start using the Trackpilots app. If employees do not install or run the Trackpilots app, their activity will not appear in the timesheet.
</Callout>

---

## 👥 Team & User Filters

Admins can filter timesheets by team, user, and date range.

| Filter Option | Description                            |
| ------------- | -------------------------------------- |
| Team Selector | View timesheets for a specific team    |
| User Selector | View individual employee timesheets    |
| Date Range    | Select daily, weekly, or custom period |

---

## ⏱️ Daily Timesheet Metrics

Each employee timesheet entry contains the following metrics:

| Metric                  | Description                              |
| ----------------------- | ---------------------------------------- |
| In Time                 | Employee first login time                |
| Out Time                | Employee last logout time                |
| Total Worked (hrs)      | Total session duration                   |
| Active Work (hrs)       | Time spent actively working              |
| Idle Work (hrs)         | Time without keyboard/mouse activity     |
| Productive Work (hrs)   | Time spent on productive apps/websites   |
| Neutral Work (hrs)      | Time spent on neutral apps/websites      |
| Unproductive Work (hrs) | Time spent on unproductive apps/websites |

## 📊 Weekly Summary Metrics

Trackpilots automatically calculates weekly summary statistics for each employee.

| Metric             | Description                    |
| ------------------ | ------------------------------ |
| Avg In Time        | Average login time             |
| Avg Out Time       | Average logout time            |
| Total Worked (hrs) | Total hours worked in the week |
| Active Work (hrs)  | Total active hours             |
| Idle Work (hrs)    | Total idle hours               |
| Productive (hrs)   | Total productive hours         |
| Neutral (hrs)      | Total neutral hours            |
| Unproductive (hrs) | Total unproductive hours       |

---

## 📈 Productivity Indicators

* 🟢 **Productive:** Work-related apps and websites
* 🟡 **Neutral:** Browsing or research activity
* 🔴 **Unproductive:** Social media or non-work activity
* ⚪ **Idle:** No activity detected
