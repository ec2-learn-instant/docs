The **Productivity Analysis** page helps managers and employees understand how time is spent across productive, neutral, unproductive, and idle activities. This data enables data-driven coaching and performance improvement.

---

## 📋 Productivity Metrics Explanation

| Metric                 | Description                             |
| ---------------------- | --------------------------------------- |
| **Total Worked (hrs)** | Total time the system was active        |
| **Active (hrs)**       | Time user was actively working          |
| **Idle (hrs)**         | Time without keyboard/mouse activity    |
| **Productive (hrs)**   | Time spent on productive apps/websites  |
| **Neutral (hrs)**      | Time spent on neutral apps/websites             |
| **Unproductive (hrs)** | Time spent on distracting apps/websites |
