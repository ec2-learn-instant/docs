# Employee Tracking API

## Start Work Session

**POST** `/api/v1/work/start`

### Request Body
```json
{
  "employeeId": "emp_123",
  "projectId": "proj_456"
}

{
  "status": "started",
  "sessionId": "sess_789"
}
