# 🧭 Overview

The **Trackpilots Dashboard** is the central command center for managers and admins to monitor employee activity, productivity, and time tracking in real time.

It includes:

* 👥 Team status overview
* ⚡ Live employee activity
* 📊 Productivity metrics
* 📸 Recent screenshots
* 🕒 Work time analysis
* 🌐 App & website usage
* 📈 Weekly productivity trends

# ⚡ Employee Recent Activity (Live)

This section shows **real-time employee status**.

### Status Types

* 🟢 **Online** – Employee is connected
* 🕒 **Work Mode** – Actively working
* 🔒 **Privacy Mode** – Tracking paused
* 📴 **Offline** – Not connected

### Displayed Information

| Field             | Description         |
| ----------------- | ------------------- |
| Associate Name    | Employee name       |
| Company           | Organization name   |
| Connection Status | Connected / Offline |
| Last Active Time  | Last heartbeat time |

# 📊 Work Summary

The **Work Summary** provides aggregated productivity metrics for a selected date and team.

### Key Metrics

| Metric                  | Description                            |
| ----------------------- | -------------------------------------- |
| Total Active Members    | Employees who worked                   |
| Total Worked Hours      | Total tracked hours                    |
| Active Work Hours       | Time actively working                  |
| Productive Work Hours   | Time spent on productive apps/websites |
| Neutral Work Hours      | Neutral activity time                  |
| Unproductive Work Hours | Time spent on distractions             |
| Idle Work Hours         | Inactive time                          |
| Productive Efficiency   | % productive time                      |
| Neutral Efficiency      | % neutral time                         |
| Unproductive Efficiency | % unproductive time                    |
| Websites Visited        | Total websites accessed                |
| Apps Visited            | Total applications used                |
